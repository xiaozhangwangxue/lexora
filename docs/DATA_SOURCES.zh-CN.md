# 数据来源与隐私

Lexora 只在用户点击“开始生成”后联网查询。查询内容包括用户输入的英文单词以及需要翻译的词典释义和例句。

- `dictionaryapi.dev`：英文定义、音标、例句和部分近反义词。
- `api.datamuse.com`：相关词和公开语料的词频信号。
- `api.mymemory.translated.net`：英文释义与例句的中文翻译。
- `fonts.google.com`：首次生成 PDF 时按需获取 Noto Sans SC 字体并缓存。

历史记录和生成的 PDF 默认保存在设备本地。Lexora 不要求账户，也不把用户的历史记录上传到项目自有服务器。第三方接口各自受其隐私政策和服务条款约束。

“难度”是基于公开词频和词形长度的学习级别估算，不是官方考试分级。词频值来自 Datamuse 返回的语料信号，适合排序和相对比较，不应理解为精确到特定语料库的百分比。
