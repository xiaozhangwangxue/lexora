# Lexora 1.3.0-beta

这是一个仅通过 GitHub 提供的公开测试版本，不会出现在 Lexora 应用内更新或官网稳定版下载中。

- 修复 Android 从历史或设置切换到生成记录时错误返回首页并唤起键盘的问题。
- 桌面窗口过窄、侧栏无法展开时隐藏无效的展开按钮，并优化非线性过渡。
- 新增 A4、A5、B5 纸张尺寸；PDF、DOCX 与 EPUB 按纸张和字号自动选择一至三栏。
- 小字号预设的单词标题改为 12pt，所有精细字号最低可设为 6pt。
- 修复未定义 `&nbsp;` 实体导致部分第三方 EPUB 阅读器无法打开的问题。
- 更新首次使用教程，并在 macOS DMG 中补充中英文“仍要打开 / Open Anyway”说明。

---

This public beta is distributed through GitHub only. It is intentionally excluded from Lexora's in-app updater and stable website downloads.

- Fixed Android opening Home and the keyboard when navigating from History or Settings to Generated.
- Hidden the unavailable desktop sidebar toggle in narrow windows and refined its nonlinear transition.
- Added A4, A5, and B5 page sizes with automatic one-to-three-column layouts for PDF, DOCX, and EPUB.
- Set the Small preset word title to 12pt and lowered every fine typography control to 6pt.
- Fixed the undefined `&nbsp;` entity that prevented some third-party EPUB readers from opening books.
- Refreshed onboarding and added bilingual Open Anyway guidance to the macOS DMG.
